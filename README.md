Data and code relative to the Gallucci et al. in prep manuscript about the chemoautotrophy of Ca. Sulfurimonas pluma at the Aurora vent field between the near-field and far-field in the weekly stratified Arctic Ocean. Sequences are available through the NCBI SRA archive with accession numbers PRJNAXXXXX for the amplicon data.

The repository contains the following files:

- README.md (this readme file)

- a series of .csv files to replicate the figures and analyses reported in the paper
